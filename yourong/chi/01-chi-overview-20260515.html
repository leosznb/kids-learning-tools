課文一：《賣柑者言》（劉基）🔴 文言｜僅考過1次

檔案

項目 內容
文體 寓言式諷刺文言
核心手法 借物喻理：以「金玉其外，敗絮其中」的柑，諷刺徒有其表的朝廷官員
校本筆記來源 校本課程 論四端-論語八則-買柑者言.pdf

歷年考情

年份 考了什麼
2425 Q1詞解（潰、瞽）+ Q3反諷手法（4分）：「賣柑者愈是讚美官員，愈顯得他們表裡不一，鞭撻愈辛辣」

⚠️ 從未考過的「處女角度」

角度 預測概率 說明
文言翻譯 🔴🔴🔴 極高 2425只考了詞解，翻譯完全空白，2526幾乎必出
「金玉其外敗絮其中」寓意闡釋 🔴🔴🔴 極高 全篇最核心的比喻，從未被獨立出題
賣柑者的形象分析 🔴🔴 中高 他是一個敢於直言的市井小人物，角色功能類似鄒忌
與其他諷諫文章的比較 🔴 中 可與鄒忌的進諫方式做對比

必背知識點

【詞解高頻】
  善藏柑，涉寒暑不潰    潰＝腐爛
  將衒外以惑愚瞽乎      衒＝炫耀；瞽＝瞎子
  出之燁然，玉質而金色  燁然＝光彩鮮明的樣子
  予貿得其一，剖之      貿＝買

【關鍵文句翻譯】（預測必考）
  「金玉其外，敗絮其中」
  → 外表像金玉一樣華美，內裡卻是破爛的棉絮。

【文章主旨】
  借賣柑者之口，諷刺元末朝廷官員徒有威儀外表，實則尸位素餐、毫無才幹，
  揭露「世之為欺者不寡矣」的社會現實。

【論證/寫作手法】
  ✅ 借物喻理（以柑喻人）
  ✅ 反諷（表面讚美實則鞭撻）
  ✅ 設問（「子何為者也？」→引出賣柑者的反駁）

常見失分陷阱

  • 「潰」字易誤解為「崩潰」（正確：腐爛）
  • 「瞽」字易誤認（正確：瞎子，讀音gǔ）
  • 翻譯時漏掉「出之燁然」的「之」→應譯為「拿出來」
  • 主旨題只寫「諷刺官員」→要寫出「表裡不一/尸位素餐/金玉其外敗絮其中」

課文二：《以畫為喻》（葉紹鈞）🟡 白話｜考過2次

檔案

項目 內容
文體 說理散文
核心手法 比喻說理：以繪畫之道比喻寫作之道
校本筆記來源 校本課程_單元七 說理散文.pdf

歷年考情

年份 考了什麼
2324 Q4結構作用（4分）：「作者如何使讀者不會誤把主題視作談繪畫？」
2425 Q4實用畫vs藝術畫對比（3分）：創作動機、程度之別

⚠️ 從未考過的「處女角度」

角度 預測概率 說明
「實用畫 vs 藝術畫」完整對比表 🔴🔴🔴 極高 2425只考了3分簡答，未考表格填空。這是表格題的黃金素材
「法度」在創作中的角色 🔴🔴 中高 作者認為實用畫需要熟習法度，藝術畫則需超越法度
「感興」的含義 🔴🔴 中 藝術畫的核心驅動力

必背知識點

【核心概念對比】（預測表格題素材）

| 維度 | 實用畫（圖畫） | 藝術畫（畫） |
|------|---------------|-------------|
| 動機 | 為實用目的 | 表現作者的感興 |
| 要求 | 像真、清楚、能傳達意思 | 超越形似，表現精神 |
| 與法度的關係 | 必須熟習法度 | 從法度入，從法度出 |
| 程度 | 較淺 | 較深 |

【關鍵句子】
  「題目標明『以畫為喻』，就是借圖畫的情形，來比喻文字。」
  → 反覆出現的結構提示句，證明全文是比喻說理

【文章主旨】
  以繪畫的實用畫與藝術畫之分，比喻寫作的實用文與文學創作之別，
  說明不同目的需要不同程度的技巧，但都需要從熟習法度開始。

【寫作手法】
  ✅ 比喻說理（畫 → 文）
  ✅ 對比（實用畫 vs 藝術畫）
  ✅ 首尾呼應（開頭談畫，結尾點破）

常見失分陷阱

  • 誤把文章主題當作「談繪畫」（作者已反覆提示是比喻）
  • 實用畫/藝術畫區別只答一兩點，遺漏完整維度
  • 忽略「法度」在兩類繪畫中的不同角色

【運動家風度四大內涵】

| 特質 | 說明 | 文中例子 |
|------|------|----------|

  「君子之爭」（引用《論語》「君子無所爭，必也射乎」）

常見失分陷阱

  • 「服輸」不是「認輸放棄」，而是「承認技不如人＋力求改進」

⚠️ 從未考過的「處女角度」

角度 預測概率 說明
「未盡責任之苦」的心理描寫分析 🔴🔴🔴 極高 三年集中在主旨和論證方法，未考文本細節分析
責任的種類與輕重 🔴🔴 中高 文中區分了多種責任（對自己/對他人/對社會）

必背知識點

【核心論點】
  未盡責任 → 人生最大的苦
  完成責任 → 人生最大的樂
  結論：勇於承擔責任，不要逃避

【論證方法三件套】

| 論點 | 論據 | 論證方法 |
|------|------|----------|
| 未盡責任是最大苦事 | 未盡的責任像千斤重擔壓在肩頭 | 比喻論證 |
| 未盡責任是最大苦事 | 欠債未還、受恩未報、得罪人未賠禮 | 舉例論證 |
| 未盡責任是最大苦事 | 生老病死可達觀排解，責任未盡到死也痛苦 | 對比論證 |

【關鍵句子】
  「未盡責任是人生最大的痛苦，盡責任是人生最大的快樂。」
  「處處盡責任，便處處快樂；時時盡責任，便時時快樂。」

常見失分陷阱

  • 主旨歸納只寫「責任重要」→ 要寫出「苦→樂→承擔」的完整邏輯鏈
  • 論證方法表格：比喻論證和舉例論證混淆（比喻用「像/如」、舉例是具體事件）
  • 寫作目的＝勸勉，不可只寫「分析責任」

必背知識點

【詞解高頻】（2425考過、2324考過、2223考過，以下彙總）

  朝服衣冠        服＝穿戴
  旦日            旦日＝第二天
  吾與徐公孰美    孰＝誰
  王之蔽甚矣      蔽＝蒙蔽
  四境之內莫不有求於王  莫＝沒有（人）
  時時而間進      間＝偶爾/間中

【關鍵文句翻譯】（連續兩年考翻譯，2526仍可能出）

  「令初下，群臣進諫，門庭若市。」
  → 命令剛下達時，許多大臣都來進諫，宮門和庭院像集市一樣熱鬧。

  「此所謂戰勝於朝廷。」
  → 這就是所說的在朝廷上戰勝（別國）。

【三問三答對照表】（預測必背）

| 問的對象 | 回答 | 動機 | 鄒忌的反思 |
|----------|------|------|-----------|
| 妻 | 「君美甚，徐公何能及君也」 | 私我（偏愛） |  |
| 妾 | 「徐公何能及君也」 | 畏我（害怕） |  |
| 客 | 「徐公不若君之美也」 | 有求於我 |  |

【文章主旨】
  鄒忌以自己與徐公比美被蒙蔽的親身經歷為喻，勸諫齊威王廣開言路、
  採納諫言，最終實現「戰勝於朝廷」的政治效果。

【寫作手法】
  ✅ 類比說理（家事→國事，妻妾客→宮婦朝廷四境）
  ✅ 排比（三問、三答、三賞）
  ✅ 鋪墊（層層推進，由己及人、由近及遠）

常見失分陷阱

  • 「服」字誤以為「衣服」（此處是動詞「穿戴」）
  • 翻譯「門庭若市」只寫「熱鬧」→需保留「像市場一樣」的比喻
  • 人物分析只說「勇敢進諫」→要寫「有自知之明（照鏡認清自己不如徐公）＋善於類比說理」
  • 「戰勝於朝廷」解釋不能只說「打勝仗」→是「不戰而使他國臣服」

5.2 三類必殺題型 → 答題公式

題型A：段落劃分＋內容大意（必出，6-12分）

【答題公式】
  第一部分（第X-Y段）：【主題詞】＋【作者做了/說了什麼】
  第二部分（第X-Y段）：【主題詞】＋【論證/展開】
  第三部分（第X-Y段）：【主題詞】＋【結論/深化】

  答題技巧：先找文章的中心句/過渡句，它們通常就是段落分界線。

題型B：論證方法＋作用（必出，2-4分）

【五種論證方法速查】

| 方法 | 判斷標誌 | 作用句 |
|------|----------|--------|
| 舉例論證 | 人名/事件/故事 | 「以具體事例證明…增強說服力」 |
| 比喻論證 | 「像/如/猶如」 | 「以生動比喻說明…使抽象道理具體化」 |
| 對比論證 | 正反/前後/有無 | 「通過對比突出…使觀點更加鮮明」 |
| 引用論證 | 引號/名言/經典 | 「引用權威言論證明…增加可信度」 |
| 類比論證 | A和B並列比較 | 「以熟悉事物類比陌生事物…使讀者易於理解」 |

題型C：結構作用（高頻，4-5分）

【萬能結構句】

  承上啟下：
  「本段/句在結構上起承上啟下的過渡作用。『承上』總結了上文______的內容；
   『啟下』引出下文______的論述，使文章結構嚴密，層次分明。」

  首段作用（如適用）：
  「開門見山提出中心論點，統攝全文／設置懸念引起讀者興趣。」

  末段作用（如適用）：
  「總結全文，呼應開頭，深化主題。」

題型D：開放題（必出，3-4分）

【三分鐘高分框架】

  立場句（1句）：我同意/不同意作者的觀點。
  理由A（1-2句）：因為…（結合文章觀點）
  理由B（1-2句）：此外…（補充另一個角度）
  例子（1-2句）：例如…（具體事例，可以是學習/生活/社會）
  回扣（1句）：因此，我認為作者的說法是有道理的/值得商榷的。

題型E：選擇題（新題型，2425年出現）

【應對策略】
  - 文言虛詞選擇：先確認每個選項的詞性（連詞/代詞/語氣詞），
    再代入原文驗證。常見混淆點：「之」作代詞 vs 作助詞。
  - 內容理解選擇：用「排除法」先刪去明顯錯誤選項，
    再比較剩餘選項，回原文找關鍵句對照。
  - 絕不可「憑感覺」選，必須在原文找到確切依據。

題型F：成語/詞語解釋（必出，2-4分）

【答題公式】
  ① 先解釋成語的字面意思
  ② 再指出在文中的具體含義
  ③ 說明該詞的表達效果（如適用）

  示例（2324「令初下」）：
  → 「令初下」指命令剛剛下達的時候。
    在文中形容齊威王納諫後，群臣爭相進諫的熱烈場面。

5月連續三週議論 考前衝刺方向 = 議論，暗示議論題分值或難度上升

9.6 三類命題寫作 — 答題框架

題型A：記敍＋倒敍法（必出，選做率最高）

【萬能結構 — 倒敍法五段式】

  第一段（倒敍開場）：
    先寫結局／高潮場景，用題目提供的首句開頭。
    → 「看到媽媽的驚喜反應，我感到自己所做的一切都是值得的。」

  第二段（起因）：
    倒回事件起點，交代為何要做這件事。
    → 媽媽生日將至／媽媽一直想要某物但捨不得買…

  第三段（經過）：
    詳細記述努力的過程。可寫困難和如何克服。
    → 省零用錢／偷偷準備／遇到挫折…

  第四段（高潮 — 與第一段呼應）：
    寫送出驚喜的瞬間，聚焦親人的反應。
    → 運用肖像描寫（表情）、行動描寫（動作）、語言描寫（說了什麼）

  第五段（體會）：
    抒發感受＋帶出深刻的體會。
    → 不只是「我很開心」，要寫「我明白了親情的價值/付出的意義…」
評分關鍵 高分的秘密
必須用倒敍法 不用的扣結構分（1等）
第一人稱 必須用「我」
親人反應要具體 面部表情＋動作＋語言，至少兩種描寫手法
體會要深刻 不能只寫「我很開心」，要寫明白了什麼道理

題型B：借景抒情（必出）

【萬能結構 — 情景交融四段式】

  第一段（入景）：
    交代時間、地點、天氣。「清晨五點，我獨自走上太平山…」

  第二段（繪景）：
    從遠到近、從靜到動、從整體到細節描寫景色。
    運用：視覺（顏色、光線）、聽覺（鳥鳴、風聲）、嗅覺（草香）

  第三段（融情入景）：
    將個人情感投射到景物上。同樣的景，因為心情不同而看起來不同。
    → 心情愉悅時：「晨光溫柔地撫摸大地」
    → 心情沉重時：「晨光冷冷地刺穿霧氣」

  第四段（點題＋感悟）：
    回扣題目的關鍵詞（如「不一樣」），寫出景色如何觸動你、讓你明白了什麼。
致命錯誤 後果
通篇寫景、沒有抒情 低分（內容不高於16分）
通篇記事、沒有寫景 喧賓奪主，降檔
沒有點出景色如何「難忘」／「不一樣」 扣題不緊

題型C：議論文（必出）

【萬能結構 — 立場議論五段式】

  第一段（引論 — 約80字）：
    引用題目說法 → 闡釋關鍵詞 → 明確表態。
    → 「我認為能者應該多勞，原因有三。」

  第二至四段（本論 — 每段約120字）：
    論點1：能者多勞可以提升整體效率（+例子）
    論點2：能者多勞是一種社會責任（+例子）
    論點3（駁論）：有人說多勞會令能者過勞，但…
    → 每段必須有：論點句＋論據＋論證方法

  第五段（結論 — 約80字）：
    總結立場＋展望／呼籲。
評分檔次 要求
7-8檔（28分） 立場鮮明＋3項論據＋不同論證方法＋駁論
5-6檔（20-24分） 立場明確＋2項論據＋基本論證方法
4檔（16分） 立場尚清晰＋1項論據

9.7 卷二時間分配鐵律

  1小時15分鐘 = 75分鐘

  ┌─────────────┬─────────────┬──────────┐
  │  審題+選題   │  實用文寫作  │ 命題寫作  │
  │  5分鐘       │  20分鐘      │ 50分鐘    │
  └─────────────┴─────────────┴──────────┘

  實用文20分鐘分配：
    2分鐘 → 圈畫題目關鍵資訊（日期、人名、活動名）
    3分鐘 → 搭格式骨架（標題、組織、署名）
    12分鐘 → 填充內容
    3分鐘 → 檢查格式（日期、人名、置中）

  命題寫作50分鐘分配：
    3分鐘 → 快速閱讀三題，勾出最有把握的一題
    5分鐘 → 簡短大綱（5個關鍵詞即可）
    38分鐘 → 寫作（目標700-800字）
    4分鐘 → 檢查（錯別字、字數、是否扣題）

9.8 實用文＋命題寫作 — 十大注意事項總匯

🛑 實用文五誡

# 誡律 說明
1 不可混淆格式 會議紀錄不可寫成書信（加「敬啟者」即扣1分）；報告不可漏「改善建議」
2 日期必須完整 XXXX年XX月XX日，缺任何一項扣0.5分
3 人名照抄不誤 題目給的名字必須原文照抄，不可簡稱或改名
4 正文點列≠縮寫 每個點必須是完整句子，不可只寫關鍵詞
5 不要自由創作 所有內容必須來自題目提供的資料，添加虛構資訊視為錯誤

🛑 命題寫作五誡

# 誡律 說明
6 字數就是分數 <600字內容自動封頂28分；建議寫700-800字為安全區
7 必須扣題 題目關鍵詞（如「倒敍」「借景抒情」「不一樣」）必須在文中明確體現
8 首句不可改 如果題目給定首句，必須原句抄入，不可改動
9 立場貫徹始終 議論文中途不可搖擺；記敍文人稱不可中途切換
10 五段為宜 不要只寫三段（太單薄），也不要寫七八段（太零碎），五段最穩

📝 加分技巧

技巧 適用題型 操作
修辭點綴 全部 每200字至少用一個修辭（比喻/排比/反問），但不要堆砌
首尾呼應 全部 結尾回扣開頭的一句話或一個意象，結構分+1
具體數字 記敍/報告 「很多人」→「超過八成五」；「等了很久」→「足足等了兩個小時」
駁論加分 議論文 必寫一段駁論，這是7-8檔的關鍵區分點
五感描寫 借景抒情 視覺、聽覺、嗅覺至少用兩種
