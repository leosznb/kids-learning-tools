Key Formula Arsenal 核心公式武器库

📐 Indices · Laws 指数定律
aᵐ × aⁿ = aᵐ⁺ⁿ
aᵐ ÷ aⁿ = aᵐ⁻ⁿ
(aᵐ)ⁿ = aᵐⁿ
a⁻ⁿ = 1/aⁿ
a^(m/n) = ⁿ√(aᵐ) NEW
💹 Compound Interest 复利息
A = P(1 + r%)ⁿ
P = principal 本金
r% = interest rate per period
n = number of periods
Half-yearly: r% ÷ 2, n × 2
📏 Coordinate Geometry 坐标几何
d = √[(x₂−x₁)² + (y₂−y₁)²]
slope m = (y₂−y₁)/(x₂−x₁)
Midpoint = ((x₁+x₂)/2, (y₁+y₂)/2)
Section: (nx₁+mx₂)/(m+n)
🔺 Trigonometry · Basic 三角基础
sin θ = opp/hyp
cos θ = adj/hyp
tan θ = opp/adj
sin²θ + cos²θ = 1
tan θ = sin θ / cos θ
🔺 Special Angles 特殊角
30°45°60°
sin1/21/√2√3/2
cos√3/21/√21/2
tan1/√31√3
🔺 Sine & Cosine Formula NEW
a/sin A = b/sin B = c/sin C
a² = b² + c² − 2bc·cos A
Area = ½ab·sin C NEW
📐 Pythagoras 勾股定理
a² + b² = c² (∠C = 90°)
Converse: if a²+b²=c² → ∠C=90°
📦 Mensuration 求积法
Cone V = ⅓πr²h
Sphere V = ⁴⁄₃πr³
Cylinder V = πr²h
Pyramid V = ⅓ × base area × h
Area ratio = k², Vol ratio = k³
📊 Statistics 统计
Mean x̄ = Σx/n
Median = middle value (sorted)
Mode = most frequent value
Stem-and-leaf: stem=十位, leaf=个位
🎲 Probability 概率
P(E) = n(E)/n(S)
Expected value = Σ[value × P]
Expected frequency = P × n
P(A and B) = P(A) × P(B) (independent)
📐 Triangle Centres 三角形四心
Centroid 重心 → 中线交点，2:1
Incentre 内心 → 角平分线交点，内切圆圆心
Circumcentre 外心 → 垂直平分线交点，外接圆圆心
Orthocentre 垂心 → 高的交点
📐 Similar Figures 相似形
ΔABC ~ ΔXYZ (AAA / SAS / SSS)
Length ratio = k
Area ratio = k²
Volume ratio = k³

Common Traps 常见扣分陷阱

  • 🚫 只写答案不写步骤
    后果：M 分全部丢失。Section B 必须展示公式→代入→计算→答案四步。
    ✅ 即使答案很简单也写「By the formula... = ...」
  • 🚫 因式分解不彻底
    后果：漏提公因式（如 4x²−4 = 4(x²−1) 未再做 (x+1)(x−1)）— 扣 A 分。
    ✅ 分解后自查：每个因式还能再分吗？HCF 提干净了吗？
  • 🚫 不等式乘除负数不反转符号
    后果：−2x > 6 → x < −3（不是 x > −3），整题 0 分。
    ✅ 每次遇到负系数立即画箭头提醒自己
  • 🚫 三角形四心概念混淆
    后果：把 centroid 的性质（2:1）用到 orthocentre 上 — 全错。
    ✅ 记忆口诀：Centroid→median→2:1 / Incentre→angle bisector→内切圆 / Circumcentre→⊥ bisector→外接圆 / Orthocentre→altitude
  • 🚫 Bearing 方位角混淆
    后果：把 bearing 当成普通角度，没从正北顺时针量。
    ✅ Bearing = clockwise angle from North (000° to 360°)，三位数表示
  • 🚫 复利息 compounding period 搞错
    后果：半年复利时忘记 r% ÷ 2 和 n × 2。
    ✅ 审题圈出 compounding period，换算 r 和 n
  • 🚫 坐标几何斜率公式分母为零
    后果：竖直线 m = undefined，用了 (y₂−y₁)/(x₂−x₁) 直接除零。
    ✅ 先判断 x₁ = x₂？→ 垂直线，m undefined
  • 🚫 Similar Solids 面积比/体积比混淆
    后果：length ratio = 2 → area ratio 写成 2（应该是 4），volume ratio 写成 4（应该是 8）。
    ✅ 面积比 = k²，体积比 = k³ — 双重确认指数
  • 🚫 新考点 ½ab sin C 用错条件
    后果：不知道什么时候用这个公式，乱套。
    ✅ 当已知两边及其夹角时使用；如果知道三边，先用 Cosine Formula 求角
