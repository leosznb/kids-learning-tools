✏️ -ing 形式拼寫規則（三條鐵律）

  • 規則① 去 e 加 -ing：以不發音的 -e 結尾 → 去掉 e 再加 -ing。
    例：make → making write → writing arrive → arriving decide → deciding move → moving use → using prepare → preparing queue → queuing
  • 規則② 雙寫尾輔音：重讀閉音節（短元音 + 單輔音）→ 雙寫最後一個輔音字母再加 -ing。
    例：run → running stop → stopping sit → sitting get → getting swim → swimming plan → planning pop → popping begin → beginning forget → forgetting travel → travelling（英式）
  • 規則③ -ie 變 -ying：以 -ie 結尾 → 變 ie 為 y 再加 -ing。
    例：lie → lying die → dying tie → tying
  • ⚠️ y 結尾不變：以子音+y 結尾 → 保留 y 直接加 -ing（不是變 i 加 -es 的規則！）。
    例：study → studying（不是 studing！） fly → flying try → trying worry → worrying

🚫 不能加 -ing 的動詞（Stative Verbs 靜態動詞）

  • 這些動詞描述狀態而非動作，不能用進行式（be + -ing）：
  • 感官：see, hear, smell, sound, notice
  • 心理：know, believe, think (認為), understand, remember, forget
  • 情感：like, love, hate, want, need, wish
  • ❌ 錯誤：I am knowing the answer. → ✅ 正確：I know the answer.
  • ❌ 錯誤：She is wanting some water. → ✅ 正確：She wants some water.

🔊 過去式 -ed 發音規則（三種讀音）

  • /t/ 清輔音後：動詞原形以 /p, k, f, s, ʃ, tʃ/ 結尾 → -ed 讀 /t/
    例：helped liked watched rushed dressed
  • /d/ 濁音後：動詞原形以母音或濁輔音結尾 → -ed 讀 /d/
    例：played called arrived enjoyed borrowed
  • /ɪd/ t/d 後：動詞原形以 /t/ 或 /d/ 結尾 → -ed 讀 /ɪd/
    例：wanted needed started decided invited
  • 💡 口訣：清清濁濁元也濁，t/d 之後讀 id
