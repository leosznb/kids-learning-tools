两地课标差异的根源、目的与价值判断 · 语文

接续《DSE_vs_内地_语文课程标准难度对比.md》的分析——前篇揭示了 DSE 写作更深、内地背诵更深的「互换领先」格局，以及 DSE 的十二年渐进链和工具性文言能力 vs 内地的经典语料积累和 72 篇背诵机制。在此基础上追问：为什么会有这些差异？各自想达成什么？孰好孰劣？

分析日期：2026-06-08

一、差异根源：一种语言，两种处境

前面六科的差异都可以追溯到英式 vs 苏式的教育传统。语文的差异追不到那里——因为两边都在教同一种母语。差异的根源不在「谁继承了谁的传统」，而在这种语言在两地的社会处境完全不同。

1.1 香港中文：「母语」在英语世界中的位置

在香港 EMI 学校（如荃湾官立中学），中文处于一个独特的悖论中：

  • 它是大多数学生的母语——在家说粤语、从小用中文读写
  • 但它是学校的第二语言——数学、物理、化学、生物、世界历史等科目全部用英语授课（中史科通常以中文授课）

这种处境塑造了 DSE 中国语文的深层性格：它必须向学生证明「中文是有用的」。在一个英语主导的学术和职业环境中，如果中文科只教「文化修养」而不教「实用技能」，学生凭什么重视它？

这就解释了 DSE 中文四件事的深层逻辑：

1. 实用写作的独立考核：书信、报告、建议书、演讲辞、会议纪录——这些不是「文学」，是职场生存技能。在一个英语主导的职场上，一个香港学生未来用中文写正式文书的机会比内地学生少得多——正因如此，课程更需要确保他在高中阶段掌握这项技能，而不是指望他在工作中自然习得。
2. 口头汇报的 SBA：在一个双语社会里，「能用中文做一个有说服力的口头报告」不是理所当然的能力——尤其是当学生已经习惯了用英语做 presentation 时。SBA 的 15%（含口语沟通和阅读汇报）是对「中文综合运用」的制度性保障。
3. 九个范畴的「全人教育」框架：品德情意、思维、语文自学——这些范畴在中文课中承担了一个额外的功能：在一个全英语的学术环境中，中文课是学生唯一用母语讨论人生价值、道德判断、文化认同的空间。这不是「中文科的野心」，是中文科在 EMI 体系中的补偿性角色。
4. 不要求机械背诵：在内地，「背诵 72 篇」之所以可行，是因为语文课时充足、应试压力大、且传统文化对背诵记忆的重视为这一方法提供了社会认同。在 EMI 学校，学生的学术精力已被英语科目大量占用——要求他们额外背诵 72 篇古诗文是不现实的。DSE 的策略是「用理解替代记忆」：文言文考核的是独立阅读能力，不是默写。

1.2 内地语文：母语作为国家认同的载体

在内地，语文课的地位完全不同：

  • 它是唯一的官方语言的课程——没有英语来竞争「学术语言」的地位
  • 它是文化传承的核心渠道——在一个没有宗教课的课程体系中，语文与思想政治课共同承担了价值观教育的功能，其中语文更侧重文化传承和审美熏陶
  • 它是国家认同的重要载体——「文化自信」不是一句口号，是课程设计的底层逻辑

这就解释了内地语文四件事的深层逻辑：

1. 72 篇推荐背诵：这不是「落后的教学方法」——是中国传统语文教育的核心方法论。两千年来，「背诵经典」一直是中国读书人的入门功夫。2017 版课标将 72 篇古诗文列为背诵推荐篇目（实际默写考查范围因省份和卷种略有差异），不是因为没有更好的方法，而是因为「脑子里有一批经典文本」本身就是课程目标之一——它培养的是「文化记忆」，不是「记忆技巧」。
2. 18 个学习任务群的广度：整本书阅读、当代文化参与、跨媒介阅读、中华传统文化经典研习、中国革命传统作品研习、外国作家作品研习、科学与文化论著研习、跨文化专题研讨、学术论著专题研讨……这个覆盖面的野心是：让一个中国高中生在语文课上接触人类文明的主要文本类型——从《红楼梦》到《乡土中国》、从革命文学到外国经典、从学术论著到新媒体文本。
3. 整本书阅读的必修地位：《红楼梦》和《乡土中国》不是「推荐读物」，是必修学分。这背后的信念是：碎片化的阅读无法培养完整的人格——只有读完一本完整的书，你才能进入一个完整的文化世界。
4. 「文化自信」作为核心素养：四大核心素养中，「文化传承与理解」不是一个「附加项」——它是与「语言建构与运用」平起平坐的核心维度。课程标准明确要求「理解和借鉴不同民族和地区的文化，拓展文化视野，增强文化自觉，提升中国特色社会主义文化自信」。这不是政治口号——这是对「语文课到底要培养什么样的人」的根本回答。

二、各自目的：两个「学好中文是为了什么」的不同回答

维度香港 DSE 中国语文内地高中语文
对「学好中文是为了什么」的回答中文是你在这个双语社会中保持母语能力的工具——能写、能说、能读、能分析、能用中文做事中文是你作为中国人的文化根基——它承载你的历史记忆、道德资源和审美能力
核心叙事「九个范畴均衡发展」——语文是全人教育：听说读写 + 文学 + 文化 + 品德 + 思维 + 自学「四大核心素养」——语文是文化传承：语言 + 思维 + 审美 + 文化
典型学生画像一个能在 2h15min 内写一封得体书信和一篇深度议论文、能用中文就阅读书目做一次有见解的口头报告、能独立阅读一篇没见过的文言文的双语者一个脑子里有 72 篇经典、读完整本《红楼梦》、能用中文分析学术论著、对中外文化有基本了解的「文化人」
社会功能在英语主导的学术和职业环境中，维持和深化学生的母语能力和文化认同在一个统一的语言文化体系内，建立学生的文化自信和国家认同
隐藏假设中文的高级学术和书面能力在 EMI 环境中需要刻意培养——如果课程不强制要求，学生可能不会主动发展这些高阶能力中文是「理所当然」的语言——课程不需要证明中文的「实用性」，它需要证明中文的「深度」
文言文的目标「给你一篇没读过的文言文，你能独立读懂」——工具性能力「72 篇经典在你的脑子里」——文化积累

三、孰好孰劣？

语文是七科中「价值判断维度最复杂」的一科

科学科目（物理/化学/生物）的「好劣」可以根据「这套训练对接什么大学专业和职业路径」来判断。但语文不同——它关心的不是「能做什么」，而是「是什么样的人」。这使得价值判断变得极其复杂——它不是「孰好孰劣」的问题，而是「你希望语文课培养什么样的人」的问题。

DSE 做得更好的四件事

1. 写作的实战深度——「能用中文做事的成年人」
DSE 中文写作卷 2h15min、45% 权重、两篇不同类型的完整文章（实用文 + 命题作文）。这不是「作文考试」，是「成年人在真实世界中使用中文的能力测试」。一个能写出一封得体正式书信（或报告/建议书）和一篇文章逻辑清晰、论据充实的议论文的高中生，已经具备了在任何中文工作环境中独立处理书面沟通的能力。
2. 口头汇报的 SBA——「能说」不是天生的
在一个英语主导的学校环境中，「能用中文做一个有见解的公开汇报」不是自然会发生的事。SBA 的 15%（含阅读汇报的书面报告 + 口语沟通）是对「中文口语与书面表达」的制度性保障——它确保了即使在一个全英语的学术氛围中，学生仍然拥有「用母语深度表达」的经验。
3. 文言的「工具性」路线——独立阅读能力的长期价值
香港用十二年渐进链条（40 篇小学 + 25 篇初中 + 指定篇章高中）培养的「独立阅读浅近文言文的能力」，是一套可迁移的技能。十年后，当这个学生想读《世说新语》《聊斋志异》原文或任何一篇唐宋散文时，他不需要依赖白话翻译——他有自己的阅读工具。内地的 72 篇背诵给你的是「你记得的 72 篇」，香港的工具性训练给你的是「你可以读任何一篇」。
4. 九个范畴的「全人」视角——语文不止是语文课
品德情意、思维、语文自学——这三个范畴将中文课变成了在 EMI 体系中唯一的价值观和思维方法训练场。物理化学教你怎么理解世界，中文教你怎么做一个完整的人。这种「补偿性功能」不是设计缺陷，是在特定教育生态中的巧妙定位。

内地做得更好的四件事

1. 文化深度的系统性——「一个中国人的文化底盘」
72 篇背诵 + 整本书阅读（《红楼梦》《乡土中国》）+ 18 个任务群覆盖古今中外——这个体系培养的不是「会写文章的人」，而是「有文化底盘的人」。在理想情况下，十年后一个内地高中毕业生可能忘了怎么解二次方程，但《赤壁赋》的豁达、《出师表》的忠诚、《红楼梦》的人情——这些文本有望变成他理解世界的参考框架。你不能「用」这些知识去做任何具体的事情，但它们在每一个需要判断、需要安慰、需要意义的时刻默默工作。
2. 整本书阅读——对抗碎片化的最后堡垒
在一个被社交媒体和短视频格式化的时代，内地课标把「整本书阅读」作为必修学分，是教育史上罕见的「逆潮流」举动。读完《红楼梦》不是「知道了故事情节」，是在没有推送、没有摘要、没有三分钟解说的情况下，用自己的专注力走完一段完整的文化旅程。这种能力在 AI 时代是无价的。
3. 18 个任务群的广度——「不让你漏掉任何一种重要的文本」
从学术论著到革命文学、从跨文化专题到新媒体文本——内地的 18 个任务群像一个「文化全餐」，确保学生在高中阶段至少「尝过」人类文明的主要文本类型。DSE 的校本课程灵活性虽然高，但也意味着一个学生可能三年没有系统接触过外国文学或学术论著——这在课程设计层面是「给了自由但可能造成盲区」。
4. 读写聚焦换来的文化深度——「不考听说」的意外收益
这看起来矛盾——内地高考没有考查听说的机制。但从另一个角度看：正因为不考听说，18 个任务群的阅读和写作内容才能做到如此丰富。课程时间的分配是一个零和游戏——DSE 花了大量时间在实用写作、口头汇报、聆听训练上，留给文学的深度阅读和文化专题的时间就少了。内地的「读写聚焦」固然牺牲了听说能力，但换来的是文化积累的深度和广度。

四、不是孰好孰劣，而是「语文课该做什么」的两种回答

这是七科对比中最微妙的一科。科学科目的价值判断可以靠「对接什么大学专业」来裁决，历史的可以靠「世界史缺失是不是短板」来评判。但语文——

语文科的价值判断，最终取决于你相不相信「文化记忆」是有内在价值的。

如果你相信：一个受过教育的人应该在脑子里有一批经典文本——不是因为他需要「用」它们，而是因为这些文本构成了他理解世界的参考框架——那么内地的 72 篇背诵和整本书阅读是无可替代的。

如果你相信：一个受过教育的人应该能用母语清晰、有力、得体地表达自己——不是为了考试，而是因为在任何语言社会中，这是公民的基本素养——那么 DSE 的写作深度和实用写作训练是无可替代的。

没有完美的答案。最好的语文教育，恐怕是既有内地的文化底盘，又有 DSE 的实战能力——脑子里有《赤壁赋》，笔下能写建议书，口头能做好 presentation。

适配不同路径

  • 如果你想让孩子有一个深厚的中华文化底子：内地的 72 篇背诵 + 整本书阅读是最直接的路径。内地的义务教育阶段也有系统的古诗文诵读（小学 75+ 篇、初中约 60 篇），但高中阶段的 72 篇背诵是集中强化的核心机制。DSE 的 65+ 篇十二年渐进链条也是一种「润物细无声」的积累方式。
  • 如果你想让孩子具备实用的中文沟通能力（写作+口头）：DSE 的写作深度 + 实用写作 + SBA 口头汇报无可替代。内地高考作文虽然也重视议论能力，但在实用文和应用场景上差距明显。
  • 如果你的孩子在 EMI 学校（如荃官）：DSE 中文是他们的必考科目——写作 45% 权重是最大的挑战。但 EMI 环境给了他们一种独特的双语思维优势——中文和英文是两个并行的「操作系统」。

五、对海纳有容的启示

他们是 EMI 学校的学生。中文对他们来说，是在英语的汪洋大海中守住的母语岛屿。DSE 中国语文的设计恰好支持了这种处境：它不要求海量背诵（因为时间和精力已被英语科目分流），但要求深度写作和实战表达能力。

他们需要在 DSE 中文中刻意培养的，不是「背更多经典」，而是「写得更好、说得更好」——

因为在他们未来的生活中，英语会帮他们读论文、写报告、做国际 presentation。但中文——在家庭里、在朋友圈中、在面对中华文化的任何时刻——是他们「做自己」的语言。DSE 中文的 45% 写作权重不是在考他们的记忆力，是在考他们：你能不能用你最熟悉的语言，表达你最真实的想法？

语文的独特位置：它是七科中唯一一科两边都在教「同一种母语」但给出了截然不同回答的科目。差异的根源不是「英式 vs 苏式」，而是中文在香港和内地完全不同的社会处境——在香港是「在英语主导环境中需要刻意保护高级能力的母语」，在内地是「理所当然的国语」。这种社会处境的差异，比任何教育哲学的差异都更根本。

七、总结

一句话核心结论：内地语文培养「文化人」（72 篇经典文化记忆 + 整本书阅读的精神家园），DSE 中文培养「语文人」（写作深度 + 实用能力 + 口头表达）——差异根源是中文在两地的社会处境不同。
详细说明：内地语文课的野心是「给你一个文化底盘」——72 篇经典是你的文化记忆，整本《红楼梦》是你的精神家园，18 个任务群带你走遍人类文明的主要文本形态。DSE 中国语文的野心是「给你一套实战能力」——2h15min 内写两篇完整文章，用中文做一个有见地的口头报告，独立阅读任何一篇没见过的文言文。前者培养的是「文化人」——一个在精神上有归属的人；后者培养的是「语文人」——一个能用母语清晰有力地做事的人。两者没有高下——只取决于你站在哪里看中文：是站在一个以中文为唯一文化载体的社会中，还是站在一个以英文为学术语言、中文的高级能力需要刻意培养和深化的双语岛上。
