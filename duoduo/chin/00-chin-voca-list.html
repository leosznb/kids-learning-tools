⚠️ 特別考點① 結構助詞「的 / 地 / 得」（U5 語基重點）

助詞用法例句
的用在名詞前面美麗的花朵 / 媽媽的手機 / 溫暖的陽光
地用在動詞前面慢慢地走 / 認真地學習 / 快樂地唱歌
得用在動詞/形容詞後面跑得很快 / 高興得跳起來 / 寫得很漂亮

⚠️ 特別考點② 文言虛詞 + 否定詞（U7 語基重點）

文言虛詞

虛詞用法例句
其「他的/它的」或「那個」其喙 = 牠的嘴巴；其人 = 那個人
之「的」或「它」（代詞）鷸之喙 = 鷸的嘴巴；漁夫得而擒之

文言否定詞

否定詞白話意思例句
莫沒有人 / 沒有東西莫能陷也（《自相矛盾》）
無沒有無不陷也
弗不弗能應也
