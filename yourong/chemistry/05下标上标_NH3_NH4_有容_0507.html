下标 & 上标 — 有容专讲

用 NH₃ 和 NH₄⁺ 一次看懂，永远不混
中性分子
NH₃

氨分子 · ammonia

阳离子
NH₄⁺

铵离子 · ammonium ion

两个家伙，长得像。一个中性，一个带电。
区别就在 下标的数字 和 右上角有没有那撮电荷。

一、拆开看 — 两个位置，两种含义

NH₃

右下角 3：3 个 H 原子
右上角空着：不带电

N 原子 1 个 + H 原子 3 个 → 中性分子

NH₄⁺

右下角 4：4 个 H 原子
右上角 +：带 1 个正电（1 省略）

N 原子 1 个 + H 原子 4 个 → 阳离子，1+

📍 核心规则（就两句）

🔵 右下角 = 数人头 —— 告诉你「有几个这样的原子」

🔴 右上角 = 数电荷 —— 告诉你「带多少电」

二、为什么 NH₃ 加一个 H 就变成了 NH₄⁺？

这是理解 上标从哪来 的关键——

NH₃
⬆ 中性
+
H⁺
⬆ 带 1+
→
NH₄⁺
整体带 1+

N 原子上有一对孤对电子（lone pair），像一只手伸出去拉住一个 H⁺。
H⁺ 自己带一个正电荷，拉进来之后，电荷归整个离子团所有。

🔵 下标从 3 → 4
多了一个 H 原子
🔴 上标从无 → +
多了一个正电荷

道理就是这么简单：多一个 H⁺，既多一个原子（下标变），也多一个正电荷（上标变）。

三、推翻最常见的写法错误

错误写法为什么错正确写法
NH⁴⁺ 4 是 H 的个数，必须写在 右下角
写在右上角就变成「带 4 个正电」了
NH₄⁺ 只带 1 个正电，不是 4 个！		 NH₄⁺
NH₄⁺¹ 1 正电荷，数字「1」省略不写
写了反而错		 NH₄⁺
NH₄¹⁺ 电荷标在右上角，数字在左、符号在右
1 省略不写所以看不到，但 2− 是 2 左 − 右		 SO₄²⁻

四、用学会的眼再看这些

化学式右下角（下标）右上角（上标）
H₂O2 → 2 个 H空 → 中性分子
CO₂2 → 2 个 O空 → 中性分子
SO₄²⁻4 → 4 个 O2− → 2 个负电
NO₃⁻3 → 3 个 O− → 1 个负电（1 省略）
Al³⁺无 → 只有 1 个 Al，不写下标3+ → 3 个正电

五、口诀 — 记这两句就够

「右下计人头，右上数电荷」

—— NH₃ 的 3 在右下 = 3 个 H；NH₄⁺ 的 + 在右上 = 一个正电

「数字在左，符号在右，1 省略不写」

—— 2− 不是 −2；NH₄⁺ 不是 NH₄¹⁺

六、自测 — 写完揭答案

1. NH₄⁺ 的「4」写在哪个角？代表什么？
答案 → 右下角，代表 4 个 H 原子
2. NH₄⁺ 的「+」写在哪个角？代表什么？
答案 → 右上角，代表 1 个正电荷
3. NH₃ 为什么没有上标？
答案 → 它是中性分子，不带电
4. NH₃ 和 NH₄⁺ 的右下角数字为什么不同？
答案 → NH₃ 有 3 个 H，NH₄⁺ 多吸了一个 H⁺（变成 4 个 H）
5. SO₄²⁻ 的「2−」为什么 2 在前面？
答案 → 数字在左，符号在右。2− 不是 −2
6. NO₃⁻ 有几个 O？带几个电？
答案 → 3 个 O（下标 3）；1 个负电（上标 −）

七、跟 Cations & Anions 攻略的关联

有容学会这个之后，回头看上次的记忆攻略，三件事就全串起来了：

攻略里的规则跟下标 / 上标的关系
-ate 多氧 -ite 少氧说的就是 下标（O 的个数）不同
加 H 减一电说的就是 上标（电荷）变了：2− → 1−
同家族电荷相同SO₄²⁻ 和 SO₃²⁻ 的 上标相同（都是 2−），下标不同
NH₃ + H⁺ → NH₄⁺ 这一条看懂了，下标和上标就通透了。
右下角永远管「有几个原子」，右上角永远管「带多少电」——分别记账，各管各的。
