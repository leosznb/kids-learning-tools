Paper 2 · Writing Strategy

Format: 3 prompts. Choose ONE. ≥200 words. 21 marks (weight 60/200). 1 hour.

Recent topics (2425): Speech (STEAM education) · Feature article (wildlife protection) · Letter to editor (career choice)

Recent topics (2324): Similar pattern — 3 different text types, real-world themes.

✅ Do
  • Read all 3 prompts before choosing — pick the one you know most vocabulary for
  • Spend 5 min planning: write 3–4 paragraph topic sentences
  • Match the tone to the format (formal for letter to editor, engaging for speech)
  • Leave 5 min to proofread for grammar and spelling
  • Use 'Chris Wong' if you need a name
❌ Don't
  • Write below 200 words — automatic penalty
  • Use your real name
  • Write in point form — must be continuous prose
  • Ignore the format requirements (e.g. a speech needs a greeting)
  • Spend more than 55 min — save time for Paper 1 grammar check

Grammar Topics · Key Rules

Conditionals Type 2 & 3

Type 2 (unreal present): If I were you, I would study more.
Type 3 (unreal past): If I had studied, I would have passed.
Watch for mixed conditionals and inverted forms (Had I known…).

Gerunds vs to-infinitives

Some verbs take gerund (enjoy reading), some take infinitive (want to read), some take both with meaning change (stop to smoke vs stop smoking).

Adjective Patterns

It is + adj + for + noun + to-infinitive
It is + adj + that + clause
Common adjectives: important, necessary, difficult, possible, likely.

Inversion

Negative adverbs at sentence start trigger inversion:
Rarely do I see… · Not only does he… · Never have I… · Hardly had we…

Proofreading (consistent section, ~8 marks) — One error per numbered line. Three types: wrong word (underline + correct above), missing word (mark ∧ + insert), extra word (cross out). Train by reading the sentence aloud — your ear often catches errors before your eye does.

Reading Comprehension · Question Types

TypeMarksStrategy
Summary blanks4–5Read the summary sentence FIRST to know what info you're looking for. One word per blank, must be grammatically correct.
Find words (synonyms / meanings)3–4Scan the specified paragraph for the word. If asked for "similar meaning", the answer is in the text — don't use your own vocabulary.
MC questions4–6Eliminate 2 obviously wrong answers first. Watch for distractors that use words from the text but twist the meaning.
T / F / NG4–8True = text says it. False = text says the opposite. Not Given = text doesn't say either way. This is the hardest type — when in doubt, it's usually NG.
Referencing (what does 'it' refer to?)2–4Go back ONE sentence. The answer is usually the subject or object of the previous sentence.
Venn diagram / matching5Read all descriptions first, then sort. Items in the overlap apply to BOTH countries/people.
