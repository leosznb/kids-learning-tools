14.3 Snell's law calculation: n = sin i / sin r ① Forget to draw normal line; misidentify i / r
② Multi-layer refraction direction confusion		 ① Draw normal first, then label angles
② Apply Snell's law layer by layer independently
14.4 TIR dual conditions + critical angle sin C = 1/n Reversing direction: forgetting TIR needs dense → less dense Mantra: TIR = dense→less dense + i > C. Both conditions must hold; miss one = no TIR.
⭐14.5
Convex		 Lens formula 1/f = 1/u + 1/v + magnification + 4 ray diagrams (u>2f, u=2f, f<u<2f, u<f) ① Sign errors: real image v>0, virtual v<0. Concave f<0.
② Ray diagram: missing arrows, F/2F mislabeled		 ① Rule: real image → v positive; virtual image → v negative; concave lens → f negative
② Drill 3 standard rays to muscle memory: through centre (straight) / parallel→through F / through F→parallel
⭐14.5
Concave		 Ray diagram + image always virtual, diminished, upright Drawing concave lens like convex lens by mistake Concave lens rays: parallel→diverge (extend backward through F). Practise separately from convex.
14.2 Image properties: virtual, laterally inverted, same size, equal distance Virtual image position: marking distance behind mirror incorrectly Mirror symmetry method: object distance = image distance; line perpendicular to mirror.
14.7 RGB additive mixing vs filter subtraction + object colour Confusing addition (light mixing) with subtraction (filters) Contrast card: Light addition (RGB→white) vs Filter subtraction (red filter transmits only red, absorbs rest).
14.8 Frequency/wavelength ordering + band applications Mixing up order; confusing applications across bands Order: Gamma → X-ray → UV → Visible → IR → Microwave → Radio (high f → low f). Memorise one core application per band.
🛠️ Problem-Solving Methods — within syllabus only
#MethodWhen to UseCore StepsCommon Mistake
1 Snell's Law Calculation Find refraction angle / refractive index / speed in medium ① Draw normal ② Label i, r ③ Apply n₁ sin θ₁ = n₂ sin θ₂ ④ Solve Angles MUST be measured from normal, not from interface
2 Lens Formula Calculation Find image distance v / focal length f / magnification m ① Confirm u, f (check signs) ② 1/f = 1/u + 1/v → solve v ③ If needed m = v/u ④ Judge real/virtual by sign of v Concave lens: f < 0; Virtual image: v < 0
3 Critical Angle Calculation Judge if TIR occurs / find critical angle ① sin C = 1/n (medium → air) ② Compare i with C Must be dense → less dense, otherwise no C exists
4 Ray Diagram Drawing Find image position, size, nature (real/virtual) ① Draw principal axis + mark F, 2F ② Draw ≥2 standard rays ③ Intersection = image Arrow direction + F/2F position must be exact; virtual image = dashed lines
5 Plane Mirror Imaging Find mirror image position, field of view ① image distance = object distance (behind mirror) ② Object-image line ⟂ mirror ③ Draw reflected rays (dashed extension to image) Image is behind mirror, use dashed lines
6 Colour Prediction Predict object colour under filter / coloured light ① Light source colour → ② What the filter cuts → ③ What the object reflects → ④ What the eye sees Filter subtraction ≠ light addition; do not mix the two
