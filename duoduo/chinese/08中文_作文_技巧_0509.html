▶📐 體裁框架（三種必考體裁）

🟦 記敘文（借物抒情 / 寫人記事）— 考最多

五段式框架
第一段（開頭）：點題，交代時間/地點/人物/物件
第二段（起因）：事情是怎樣開始的？物件從哪裏來？
第三段（經過）：最精彩的部分，寫細節、寫感受
第四段（轉折/高潮）：轉折點或最深感受
第五段（結尾）：總結 + 點題 + 感悟
📝 記敍文萬能開頭
在我的記憶中，＿＿＿＿＿總是佔據着一個特別的位置。每當我看到＿＿＿＿＿，就會想起＿＿＿＿＿。

🟥 議論文（堅持、勇敢、成功）— 單元三重點

四段式框架
第一段（提出觀點）：開門見山，表明立場
第二段（論據一）：舉實例（名人故事、生活經歷）
第三段（論據二）：反面論證或另一個角度
第四段（總結）：重申觀點 + 對自己的啟發
📝 議論文萬能開頭
有人說＿＿＿＿＿，也有人認為＿＿＿＿＿。在我看來，＿＿＿＿＿。以下我將從兩方面說明我的觀點。

🟩 寓言改寫 / 讀後感 — 單元七重點

三段式框架
第一段：簡述原文故事（1-2句即可）
第二段：分析寓意——這個故事告訴我們甚麼道理？
第三段：聯繫生活——這個道理在我們生活中有甚麼應用？

▶📝 好詞好句（按主題分類）
🟥 堅持 / 努力
堅持不懈　持之以恆　不怕困難
迎難而上　半途而廢　鍥而不捨
「失敗乃成功之母」
「只要功夫深，鐵杵磨成針」
🟦 勇敢 / 面對
勇敢無畏　挺身而出　臨危不懼
鼓起勇氣　退縮逃避　挑戰自己
「真正的勇敢不是不害怕，而是害怕時仍然前行」
🟩 情感 / 感受
感動不已　心潮澎湃　熱淚盈眶
深深體會　銘記於心　難以忘懷
失望沮喪　喜出望外　激動萬分
🟨 成功 / 收穫
努力不懈　終於成功　收穫滿滿
一分耕耘一分收穫
「台上三分鐘，台下十年功」
「不經一番寒徹骨，怎得梅花撲鼻香」

▶📋 常見題目預演
題目體裁寫作要點
記一次失敗的經歷記事失敗的細節 → 當時感受 → 學到甚麼
我最敬佩的人記人外貌/事例 → 為何敬佩 → 對我的影響
一件心愛的物品借物物品來歷 → 外觀 → 與我的故事 → 意義
堅持就是勝利議論提出觀點 → 舉例(名人/自己) → 反面論證 → 總結
勇敢面對困難議論甚麼是勇敢 → 舉例證明 → 怎樣做到 → 總結
《自相矛盾》讀後感寓言簡述故事 → 分析寓意 → 生活例子
▶⚠️ 作文最易寫錯的繁體字
簡體繁體（正確）常見錯誤
经历經歷錯寫成「經曆」
坚持堅持「堅」下是土不是士
勇敢勇敢「敢」左邊不是「又」
收获收穫錯寫成「收獲」
困难困難「難」左邊是「莫」非「又」
母亲母親「親」右邊是「見」
感动感動「動」左邊是「重」
失败失敗「敗」左邊是「貝」
努力努力「努」上是「女」
因为因為「為」不是「为」
▶✅ 寫完檢查清單
  • 有沒有分段？（至少 4-5 段）
  • 每段開頭有沒有空兩格？
  • 標點符號有沒有漏？（。，！？）
  • 有沒有錯別字？（特別注意簡轉繁）
  • 句子通順嗎？讀一遍試試
  • 有沒有用上好詞好句？（至少 2-3 個成語）
  • 結尾有沒有點題？
  • 字數夠嗎？（約250-300字）
